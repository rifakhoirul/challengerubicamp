## Getting Started

### API
* Instalation
  ```sh
  cd api-mobilku
  npm install
  ```
* Seeder
  ```sh
  npm run seeder
  ```
* Start
  ```sh
  npm run dev
  ```

### Client
* Instalation
  ```sh
  cd api-mobilku
  npm install
  ```
* Start
  ```sh
  npm run dev
  ```
